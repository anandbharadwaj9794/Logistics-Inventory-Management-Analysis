# Logistics-Inventory-Management-Analysis
End-to-end Power BI project focused on logistics and inventory management. The dashboard delivers insights into inventory turnover, warehouse utilization, supplier lead time, transportation costs, and backorders to improve operational efficiency and planning.
Logistics & Inventory Management Analysis (Power BI)
 Project Overview

This project analyzes inventory and supply chain performance for a multi-category retail business (Clothing, Electronics, Furniture, Accessories).
The goal is to identify inefficiencies, reduce costs, minimize backorders, and support data-driven decision-making using Power BI.

 Problem Statement

Businesses often struggle with inventory imbalance, supplier delays, underutilized warehouses, and rising transportation costs.
This project aims to provide a centralized analytics solution to monitor inventory efficiency, logistics performance, and sales trends.

 Dataset

The dataset includes:

Order Date, Region, Category, Supplier

Units Sold, Inventory Levels, Warehouse Capacity

Lead Time, Order Status, Backorders

Transportation Costs


 Tools & Technologies

Power BI

Power Query (Data Cleaning & Transformation)

DAX (Measures & KPIs)

Data Modeling

Data Visualization

Data Preparation

Cleaned and transformed data using Power Query

Standardized date and category formats

Created calculated columns and measures

Built relationships for efficient analysis

 Key KPIs

Inventory Turnover Ratio

Days Sales of Inventory (DSI)

Warehouse Utilization (%)

Total Units Sold

Transportation Cost

Backorder Rate

 Dashboard Insights

Strong inventory turnover and low DSI indicate efficient stock movement

Warehouse utilization is low, revealing cost-optimization opportunities

Clothing category has the highest supplier lead time

Transportation costs are highest for Electronics and Furniture

Sales show a strong recovery post-2021 with steady growth till 2024

 Recommendations

Optimize reorder points to reduce overstock

Improve supplier lead times, especially for Clothing

Better utilize warehouse capacity

Optimize logistics routes to reduce transportation costs

Monitor pending backorders proactively

 Outcome

The dashboard enables:

Smarter procurement decisions

Streamlined logistics operations

Improved demand planning

Enhanced operational efficiency and customer satisfaction
